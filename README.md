EDA_Sleep_Health

🔍 Objetivo del análisis

Explorar relaciones entre variables y validar tres hipótesis principales:

El estrés afecta negativamente al sueño.

La actividad física podría mejorar la calidad del descanso.

Los trastornos del sueño podrían estar asociados a perfiles concretos (peso, estrés, actividad física).

📊 Resumen de resultados

El estrés muestra una relación clara con peor calidad y menos horas de sueño.

El insomnio aparece ligado a altos niveles de estrés.

La apnea se relaciona más con el peso corporal que con el estrés.

La actividad física podría influir positivamente (se recomienda análisis más profundo).
